Document Object Model Learning

El DOM es una interfaz de programación de aplicaciones (API) para documentos HTML y XML. Define la estructura lógica de los documentos y la forma en que se accede y manipula un documento.

El DOM representa el documento como nodos y objetos. Eso permite a los programadores agregar, eliminar y modificar partes de la página web.
